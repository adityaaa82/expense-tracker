# Expense Tracker

A simple full-stack finance tracker that helps monitor income and expenses with visual insights.

## Overview
The app lets users record, edit, and categorize transactions, then view totals and comparisons through responsive charts.  
Data is managed globally via Redux and stored on a backend powered by Node.js and MongoDB.

## Key Features
- Add, edit, and delete transactions  
- Filter by type, category, and date  
- Dynamic Pie and Bar charts for analysis  
- Responsive, minimal UI built with TailwindCSS  
- Server-generated transaction IDs  

## Tech Stack
**Frontend:** React + TypeScript + Redux Toolkit + Chart.js  
**Backend:** Node.js + Express + MongoDB  

A lightweight project focused on clarity, performance, and real-time financial tracking.
